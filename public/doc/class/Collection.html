# (String) stich(a, b)

stiches two or more paths together

Examples:

Stich two paths 

var c = new Collection("my_coll").stich("path","name"); // "path.name"

Stich three paths 

var c = new Collection("my_coll").stich("path","name","[0]"); // "path.name[0]"

Parameters:

  • a ( String ) — first string to stich
  • b ( String ) — second string to stich

Returns:

  • ( String ) — A string of a JSON PATH

# (mixed) findByPath(path)

Does a deep search and returns the object if found or else undefined

Examples:

Find name of a person 

new Collection("person",{name: "valtid"}).findByPath("[0].name");//"valtid"

Parameters:

  • path ( String ) — a JSON path to search for

Returns:

  • ( mixed ) — returns different types of undefined if nothing found
